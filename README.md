A high-performance, premium personal portfolio architected with **Next.js 15** and **TypeScript**. This project serves as a digital showcase of engineering skills, focusing on backend-leaning expertise while delivering a high-end visual experience.

## üìñ About This Project

This portfolio is built to demonstrate the intersection of robust development and polished user experience. By utilizing **Framer Motion**, the project implements physics-based layout animations, dynamic spotlight hover effects, and smooth scroll behaviors. It features a centralized data management system in `data/content.ts`, making it easily extensible and maintainable.

## üöÄ Getting Started

### Prerequisites

- Node.js 18+ 
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/anish435/anish-portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üìÅ Project Structure

- `app/`: Next.js App Router pages and global styles.
- `components/`: Reusable UI components (Hero, ProjectCard, Section, etc.).
- `data/`: Centralized content management (`content.ts`).
- `public/`: Static assets (images, icons).
